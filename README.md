# Miron.uz

Yangiliklar portali uchun tayyorlangan front-end loyiha.

## Xususiyatlar

- Bosh sahifa va kategoriya sahifalari
- Alohida yangilik sahifasi
- Izoh qoldirish
- Ko'rishlar sonini saqlash
- Admin panel
- Yangilik qo'shish, tahrirlash, o'chirish
- Izohlarni boshqarish

## Ishga tushirish

```bash
npm start
```

So'ngra brauzerda `http://127.0.0.1:3000` manzilini oching.

## Admin kirish

- Parol: `miron123`

## Eslatma

Bu loyiha backend siz ishlaydi va ma'lumotlar brauzer `localStorage` ichida saqlanadi.
